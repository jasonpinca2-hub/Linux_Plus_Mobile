# Linux+ Trainer Rev 7.0

## Added
- Mistake Notebook with automatic per-question learning history
- Attempts, correct answers, misses, last-attempt date, and correct-answer streaks
- Five-consecutive-correct mastery target
- Personal notes available from both the exam and notebook
- Smart filters for Needs Practice, all mistakes, favorites with history, mastered, and all tracked questions
- One-click practice for needs-review, mistake, and low-confidence question sets
- Correct guesses and high-confidence misses are retained as review priorities
- Manual mastery override

## Compatibility
Existing Rev 6.x favorites, confidence ratings, history, PWA data, and saved settings remain compatible.
