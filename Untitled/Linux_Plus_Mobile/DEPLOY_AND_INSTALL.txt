LINUX+ TRAINER PWA — DEPLOY AND INSTALL

IMPORTANT
A Progressive Web App cannot be installed from the iPhone Files app. It must first be hosted at an HTTPS web address.

EASIEST DEPLOYMENT: NETLIFY DROP
1. On a Mac or PC, extract this ZIP.
2. Visit Netlify Drop in a browser.
3. Drag the extracted folder (not the ZIP) onto the deployment page.
4. Netlify provides an HTTPS address.
5. Open that address in Safari on the iPhone.
6. Tap Share → Add to Home Screen → Add.
7. Launch Linux+ Trainer from the new Home Screen icon.

GITHUB PAGES
1. Create a GitHub repository.
2. Upload every file and folder from this package to the repository root.
3. In repository Settings → Pages, deploy from the main branch/root folder.
4. Open the generated HTTPS Pages address in Safari.
5. Tap Share → Add to Home Screen.

OFFLINE USE
After the first successful visit, the service worker caches the exam, images, and app files. Open the Home Screen app once while online before relying on offline mode.

UPDATES
When a newer build is deployed, close and reopen the app while online. Safari may need a refresh before the new cached version replaces the previous one.
