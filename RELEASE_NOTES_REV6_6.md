# Linux+ XK0-006 Certification Trainer — Rev 6.6

## New: Favorites Practice Sheet

- Added **Print / Save Favorites** to the Exam Setup screen.
- Builds a clean worksheet from every question saved to Favorites.
- Includes question text, answer choices, reference images, and note-writing lines.
- Optional answer key can be included at the end.
- **Print / Save as PDF** uses the browser print dialog.
- **Download HTML** saves a standalone practice sheet for offline use.
- Existing Favorites-only exam mode remains available.

All Rev 6.5 features and the existing local-storage favorites list are preserved.
